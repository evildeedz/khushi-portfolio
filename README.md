# Khushi Singh - Portfolio

A modern, responsive portfolio website showcasing UI/UX design work.

## Features

- Interactive 3D portfolio slider
- Responsive design
- Modern animations and effects
- Contact form integration
- CV download functionality

## GitHub Pages Deployment

This portfolio is automatically deployed to GitHub Pages using GitHub Actions. The deployment is triggered on every push to the main branch.

### Setup Instructions

1. Go to your repository settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "GitHub Actions"
4. The workflow will automatically deploy your portfolio

## Live Demo

Your portfolio will be available at: `https://[your-username].github.io/khushi-portfolio`
